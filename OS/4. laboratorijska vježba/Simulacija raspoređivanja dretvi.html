Simulacija rasporeðivanja dretvi

ZADATAK

Simulirati postupak rasporeðivanja dretvi u sustavu prema zadanom postupku (FIFO, prioritetno ili RR). Nije potrebno stvarati dretve radi simulacije obavljanja posla, dovoljno je ispisati kojim se redom one obavljaju (primjer ispisa je u nastavku).

Svojstva dretvi mogu se zadati na nekoliko naèina (odabrati željeni):

  • strukturom podataka (pogledati primjer);
  • unositi za vrijeme rada ili
  • naèiniti generator koji æe nasumièno stvarati dretve (s odreðenim parametrima).

Što kad se nešto dogodi "istovremeno"? Npr. kraj jedne jedinice vremena poklapa se s dolaskom nova posla.
Preporuka: najprije "obraditi" kraj, a tek onda nove dogaðaje. Npr. na kraju "sekunde" ažurirati broj dodijeljenih sekundi aktivnoj dretvi, napraviti odgovarajuæe radnje, ovisno o rasporeðivaèu, a tek potom pogledati ima li pojave nove dretve (i akcije koje to nosi).
 

Prijedlog strukture podataka te primjer ispisa

int t; //simulacija vremena (trenutno vrijeme)

struct dretva {
	int id; //1, 2, 3, ...
	int p;  //preostalo vrijeme rada
	int prio; //prioritet
	int rasp; //naèin rasporeðivanja (za opcionalni zadatak)
};


#define MAX_DRETVI	5
struct dretva *P[MAX_DRETVI]; //red pripravnih dretvi, P[0] = aktivna dretva

/* podaci o dogaðajima pojave novih poslova - dretvi */
#define DRETVI	6
int nove[DRETVI][5] =
{
	/* trenutak dolaska, id, p, prio, rasp (prio i rasp se koriste ovisno o rasporedivacu) */
	{ 1,  3, 5, 3, 1 }, /* rasp = 0 => PRIO+FIFO; 1 => PRIO+RR */
	{ 3,  5, 6, 5, 1 },
	{ 7,  2, 3, 5, 0 },
	{ 12, 1, 5, 3, 0 },
	{ 20, 6, 3, 6, 1 },
	{ 20, 7, 4, 7, 1 },
};

void ispis_stanja ( int ispisi_zaglavlje )
{
	int i;

	if ( ispisi_zaglavlje ) {
		printf ( "  t    AKT" );
		for ( i = 1; i < MAX_DRETVI; i++ )
			printf ( "     PR%d", i );
		printf ( "\n" );
	}

	printf ( "%3d ", t );
	for ( i = 0; i < MAX_DRETVI; i++ )
		if ( P[i] != NULL )
			printf ( "  %d/%d/%d ",
				 P[i]->id, P[i]->prio, P[i]->p );
		else
			printf ( "  -/-/- " );
	printf ( "\n");
}

int main () {/* napraviti! */}
